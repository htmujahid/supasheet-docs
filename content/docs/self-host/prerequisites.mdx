## Required Software

### Node.js 18+

Supasheet requires Node.js version 18 or higher.

```bash
# Check version
node --version  # Should be v18.0.0 or higher

# Install Node.js from nodejs.org if needed
```

### Docker

Docker is required for running Supabase locally.

```bash
# Check Docker installation
docker --version

# Download from docker.com if needed
```
