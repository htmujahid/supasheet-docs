## Free Trial

Your first installed project starts a **7-day free trial** with the full feature set — hosted frontend, UI builder, and AI assistant. The trial window starts when you install Supasheet into the project and is not reset by reinstalling.

When the trial ends, the hosted frontend and builder access for that project are paused. Your data is untouched: everything lives in your own Supabase project, and the Supasheet schema installed there keeps working (for example with a [self-hosted frontend](/self-host)).

## Upgrading

Hosted plans are handled personally rather than through self-serve checkout. To upgrade, add projects, or discuss custom domains:

**Contact us at [info@supasheet.app](mailto:info@supasheet.app)**

## Project Limits

Accounts start with a limit of one installed project. Upgraded accounts get higher limits and their projects are activated without a trial period.

## Leaving Is Easy

<Callout title="No lock-in by design">
The managed platform installs the same open-source migrations as the public repository, into a database you own. If you ever want to leave, deploy the self-hosted frontend against the same project (see [Self-Host](/self-host)) — your data, configuration, users, and permissions carry over as-is. **Drop** from the dashboard removes the Supasheet schema and edge functions if you want a full uninstall.
</Callout>
